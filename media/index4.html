My Snapchat




Le but du projet étais de créer une application mobile qui envoie des images a mes amis pour un certain laps de temps, puis qui seront supprimés par la suite. Ce projet devra s'appelé "My_snapchat", pour créer plus librement ce projet un API nous étais fournis pour que toutes les personnes utilisant cet Api puisse s'envoyer des photos.
